# 🚀 Project Overview

The **Personal Finance Analytics Dashboard** is a comprehensive financial management system built entirely in **Google Sheets**.

The project automates the complete financial tracking workflow—from collecting transaction data through Google Forms to generating interactive dashboards, financial KPIs, and actionable monthly insights.

Rather than serving as a simple expense tracker, the dashboard provides an analytical view of personal finances through dynamic visualizations, automated calculations, and real-time reporting.
