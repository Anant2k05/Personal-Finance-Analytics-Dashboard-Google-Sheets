# 🚀 Project Overview

The **Personal Finance Analytics Dashboard** is an end-to-end financial management system built entirely in **Google Sheets**.

It automates the complete financial tracking workflow—from collecting transaction data through Google Forms to generating interactive dashboards, financial KPIs, and actionable monthly insights.

Rather than serving as a simple expense tracker, the dashboard provides an analytical view of personal finances through dynamic visualizations, automated calculations, and real-time reporting.

# 📋 Dashboard Modules

### 📊 Dashboard Overview
Provides a complete overview of monthly financial performance through KPIs, charts, and summary cards.

### 💸 Expense Analytics
Tracks category-wise spending, budget adherence, and expense distribution.

### 💰 Income Analytics
Analyzes income sources, contribution percentages, and monthly income trends.

### 💳 Debt & Lending
Monitors outstanding debt, repayments, recovery rate, and money lent.

### 🔄 Self Transfers
Tracks transfers between accounts and automatically updates balances.

### ⚙️ Dashboard Control Panel
Allows users to configure budgets, savings goals, income targets, and reporting periods dynamically.

# ⚙️ How It Works

1. Financial transactions are submitted through a dynamic Google Form.
2. Responses are automatically stored in the transaction database.
3. Advanced spreadsheet formulas process and categorize every transaction.
4. Charts, KPIs, and dashboard components refresh automatically.
5. The dashboard generates real-time financial insights without manual calculations.
